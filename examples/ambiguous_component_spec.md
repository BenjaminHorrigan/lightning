# Centrifugal Turbopump — Preliminary Specification

**Part Number:** CTP-9X (prototype designation)
**Revision:** 1 (2026-03-30)
**Program:** (not specified — prototype for vendor quote)

## Functional Description

Single-stage centrifugal turbopump sized for high-density cryogenic fluid
service. The unit comprises a shrouded impeller, a radial-inlet volute,
a single-stage partial-admission axial turbine drive, mechanical shaft
seals, and an integral inducer. The assembly is designed for a single
shaft speed range with no throttle capability.

## Materials

- Impeller: Inconel 718
- Turbine blades: CMSX-4 single-crystal nickel superalloy
- Shaft: 17-4 PH stainless
- Bearings: Si3N4 ceramic hybrid
- Seals: Carbon-graphite face seals
