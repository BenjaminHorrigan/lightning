# Model Rocket Launch Pad — Specification Sheet

**Part Number:** MRLP-24-NFPA
**Revision:** 2 (2026-02-01)
**Program:** Educational hobby-rocket product line

## Functional Description

A portable launch pad assembly for consumer model rockets, comprising a
tripod stabilizer base, an adjustable launch rod holder (nominal 1/8 in.
and 3/16 in. rods), a blast deflector plate, and a four-wire electric
launch controller with safety key interlock. The unit supports
single-stage hobby motors from A-impulse through G-impulse class.

## Performance Parameters

| Parameter                    | Value       | Units    |
|------------------------------|-------------|----------|
| Maximum supported motor      | G-impulse   | class    |
| Launch rod length            | 36          | inches   |
| Base deployment footprint    | 24x24       | inches   |
| Control wire length          | 15          | feet     |
| Operating temperature range  | -10 to 45   | C        |

## Materials

- Tripod legs: 6061-T6 aluminum, anodized
- Rod holder: stainless steel 304
- Blast deflector: carbon steel, ceramic-coated
- Control wire: insulated 18-gauge copper
- Housing: ABS injection-molded

## Compliance and Intended Use

This launch pad is designed and marketed exclusively for consumer model
rocketry in compliance with **NFPA 1122 — Code for Model Rocketry**. All
supported motors are NAR/TRA-certified and fall within the Class-1
amateur category. No support for high-power rocketry (Class-2 or
Class-3), no provision for commercial-grade or experimental motors, and
no interfaces suitable for military or launch-vehicle applications.

## Notes

Product is classified by the manufacturer as a consumer hobby good
pursuant to NFPA 1122. The 22 CFR 121.1 Category IV(c) note excluding
NFPA 1122-compliant consumer model rocketry items from the US Munitions
List applies. Commercial availability is broad (NAR and TRA vendors
distribute equivalent launch pads nationally).
